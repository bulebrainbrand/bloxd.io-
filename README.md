# bloxd.io-建築ツール
## 概要
bloxd.ioのWorldCodeに貼り付けることでいくつかの機能が使えます
## 使えるツール
> [!IMPORTANT]
> 基本的に設置するブロックは登録ブロックリストからランダムに選ばれます
### ブロック登録
- 木の剣:クリックしたブロックを登録ブロックリストに入れます
- 石の剣:登録ブロックリストを最新のものから消します
- 鉄の剣:登録ブロックリストをクリアします
> [!tip]
> 木の剣で空中をクリックするとAirを登録できます
### 円設置
- 木の斧:クリックした地点を中心に円を描きます
- 石の斧:スニークしながらクリックで円の大きさを拡大,普通にクリックで縮小します
- 鉄の斧:クリックすると円を描く向きを変更します
> [!TIP]
> ```/circleradius 数字```あるいは```/cr 数字```でも大きさを設定できます

> [!TIP]
> ```/circleaxisnext```あるいは```/can```で鉄の斧を代用できます
### 球設置
- 木のシャベル:クリックした地点を中心に球を設置します
- 石のシャベル:スニークしてクリックで球のサイズを拡大,普通にクリックで縮小します
> [!TIP]
> ```/sphereradius 数字```あるいは```/sr 数字```でも大きさを設定できます
### 範囲設置
- 木のつるはし:範囲内すべてをしきつめます
- 石のつるはし:範囲を選択できます
### 線設置
- 木のくわ:2点間にブロックを置きます
- 石のくわ:2点を選択します
> [!WARNING]
> 線には誤差があります
### チェック柄設置
- シールドの矢:範囲内を土と砂が交互に並ぶように敷き詰めます
- 力の矢:範囲を選択します
### その他の機能
- 棒:クリックしたブロックを登録ブロックリストの中からランダムに変更します
- 負傷の矢:クリックしたブロックを空気に置き換えます
## 常時発動の機能
### ハーフブロック設置支援
ハーフブロック設置時にハーフブロックを見てるハーフブロックと同じ向きに置けます</br>
スニーク中に置くと無効になります
